⌨️ Constructor de Búsquedas JQL

Prueba a escribir tus propias consultas JQL y ve ejemplos comunes

Ejemplos de Consultas JQL
  • assignee = currentUser() AND status = "To Do"
  • project = "FINANZAS" AND priority = Highest
  • created >= -7d AND component = "Login"
  • sprint in openSprints() AND issueType = "Story"
❓ ¿Qué consulta JQL usarías para encontrar todos los bugs sin asignar en el proyecto 'WEB'?
project = WEB AND issueType = Bug AND status = Unassigned
project = 'WEB' AND issueType = 'Bug' AND assignee IS EMPTY
project = 'WEB' + issueType = 'Bug' + assignee = null
